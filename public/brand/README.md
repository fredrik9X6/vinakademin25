# Vinakademin — logos

All logos are shipped in **two formats**:
- **PNG** (4× retina) — use for web, email, social, presentations
- **SVG** with Coolvetica embedded inline (~247 KB each) — use for design tools, print, and anywhere a vector is needed

**Never set the Va mark next to the wordmark.** The Va already starts with V — placing the wordmark right after repeats the brand and looks redundant. Use one or the other.

## Wordmark — "Vinakademin"

The primary identity. Coolvetica with the brand orange gradient (`#FB914C → #FDBA75`).

| File | Use on |
|---|---|
| `wordmark-orange.png` / `.svg` | Any neutral surface (works on white, cream, dark) |
| `wordmark-ink.png` / `.svg` | Light surfaces only (#1a1a1a fill) |
| `wordmark-white.png` / `.svg` | Dark or orange surfaces (white fill) |

## Logomark — "Va"

The square monogram. Use for app icons, avatars, social squares, and any tight slot where the wordmark won't fit (under ~120 px wide).

| File | Background |
|---|---|
| `logomark.png` / `.svg` | Primary — gradient orange (alias for `logomark-gradient`) |
| `logomark-gradient.png` / `.svg` | Gradient orange + white Va |
| `logomark-dark.png` / `.svg` | #1a1a1a + gradient Va |
| `logomark-cream.png` / `.svg` | Cream paper + gradient Va |
| `logomark-white.png` / `.svg` | White + gradient Va |

## Favicon

Same Va monogram in a smaller-radius square. Use for browser tabs, OS app icons, anywhere under ~32 px. Coolvetica's letterforms stay legible all the way down to 16 px.

| File | Notes |
|---|---|
| `favicon.png` / `.svg` | Va mark on gradient orange (96-px corner radius) |
| `logomark-favicon.png` / `.svg` | Same image — duplicated under the logomark prefix for predictable naming |

## Don't

- Don't combine the Va mark and wordmark side-by-side
- Don't re-color the wordmark
- Don't re-draw the Va on a different shape
- Don't put either mark on a busy gradient, photograph, or low-contrast surface
- Min clear space around the wordmark = the cap height on all sides
