# Archive

Historical planning docs, fix summaries, and incident writeups. Kept for context (why a change was made, what the symptoms were) but **not guaranteed to reflect current code**. Many describe work that has since been superseded.

For active guidance see the top-level `README.md` and `CLAUDE.md`.

When adding new docs here, prefer a filename prefixed with the year (e.g. `2026-03-stripe-webhook-timeout.md`) so the directory sorts chronologically.
